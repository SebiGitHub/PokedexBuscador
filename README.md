Buscador de Pokémons con Kotlin

Aplicación desarrollada en Kotlin para Android (o multiplataforma, según tu enfoque), que permite buscar y explorar información de distintas especies de Pokémon. Consume datos desde una API pública externa para mostrar nombre, imagen y detalles de cada Pokémon al usuario — simulando la experiencia de una Pokédex digital. El foco está en:
- Consumo dinámico de API y parseo de datos JSON.
- Gestión eficiente de peticiones y carga de recursos.
- Interfaz de usuario simple e intuitiva que facilita búsquedas.
